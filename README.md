A Password Generator web application built using HTML, CSS, and JavaScript that generates strong and secure random passwords based on selected options like length, numbers, symbols, and uppercase/lowercase letters.

# 🔐 Password Generator

A simple and secure **Password Generator** that allows users to create strong random passwords instantly. Users can customize password length and include uppercase letters, lowercase letters, numbers, and special symbols.

This project demonstrates **JavaScript randomization, string manipulation, and DOM interaction**.

---

## 🚀 Features

* Generate strong random passwords
* Choose password length
* Include uppercase letters
* Include lowercase letters
* Include numbers
* Include special symbols
* Copy password to clipboard
* Clean and responsive UI

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Styling and layout
* **JavaScript** – Password generation logic
